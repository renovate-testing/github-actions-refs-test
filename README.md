# GitHub Actions Refs Test

Test repository for Renovate's GitHub Actions manager with:

- **Feature-oriented tags** - non-semver tags like `cargo-llvm-cov` for `taiki-e/install-action`
- **Branch references** - actions pinned to branches like `main`
- **Digest pinning** - both feature tags and branches with SHA pinning

See `.github/workflows/test.yml` for examples.
